# Olist Supply Chain Analytics Hub 🚚

A data-driven logistics optimization engine built with the Brazilian E-Commerce Public Dataset. This project leverages relational databases and an interactive user interface to analyze fulfillment bottlenecks, optimize freight costs, and evaluate customer retention.

## 🛠️ Tech Stack
* **Language:** Python
* **Database Layer:** SQLAlchemy & SQLite
* **Dashboard Interface:** Streamlit
